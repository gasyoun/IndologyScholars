# Зограф 2026: участники без указания институциональной аффилиации

Из 60 участников XLVII Зографских чтений (2026) **в опубликованной программе** ни у одного не указано конкретное учреждение — только город.

Сопоставление с историческими аффилиациями из программ предыдущих лет:

- **19 из 60** (31.7%) имеют институциональную принадлежность, зафиксированную в наших данных за прошлые годы;
- **41 из 60** (68.3%) **ни разу за все годы участия** не указывали учреждение — известен только город.
