# Reuse Rights

[Development notes](development-en.md) | [Data package](../datapackage.json) | [Citation metadata](../CITATION.cff)

This repository contains several kinds of material with different reuse
conditions.

| Material | Reuse Position |
| --- | --- |
| Repository-native code, scripts, templates, and validators | Apache-2.0, as stated in `LICENSE`. |
| Normalized metadata, derived CSV/JSON exports, SQLite records, and generated public pages | CC-BY-4.0 unless a file states otherwise. Cite the archive and keep source/provenance fields intact. |
| Cached conference programmes, quoted source snippets, and source images | Retained for verification; reuse is governed by the original rightsholders and source sites. |
| Third-party fonts, icons, authority records, and external identifiers | Governed by their upstream licences or database terms. |
| Article drafts and submission files | Scholarly manuscript material; do not reuse as dataset/code without explicit author permission. |

For reproducible reuse, cite `CITATION.cff`, record the release tag or commit,
and prefer the published file manifest in
`analytics_output/publication_file_manifest.{csv,json}` for checksums.
