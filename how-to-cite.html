⚙️ Машиночитаемые файлы и схемы

CITATION.cff
CITATION.cff — стандартный машиночитаемый файл описания цитирования для программного обеспечения и репозиториев (поддерживается GitHub и GitLab).
datapackage.json
datapackage.json — строгий манифест Frictionless Data Package, содержащий схемы данных, хэши и кодировки для всех CSV-таблиц архива.
