# Архив / Archive

[Текущая документация](../docs/README.md) | [Current documentation](../docs/README.md)

Этот каталог хранит датированные рабочие материалы и снимки состояния,
которые полезны для истории решений, но не описывают текущую публикацию.
Актуальные показатели берутся из `site_data.json` и документов в `docs/`.

This directory preserves dated working materials and state snapshots that are
useful for decision history but do not describe the current publication.
Current figures come from `site_data.json` and the maintained documents under
`docs/`.

| Каталог / Directory | Содержание / Contents |
| --- | --- |
| `plans/` | Архитектурные, SciGuide, SEO и вспомогательные планы / architecture, SciGuide, SEO, and auxiliary plans |
| `reports/` | Устаревшие аналитические снимки / superseded analytical snapshots |
| `session/` | Старые handoff и AI-state файлы / old handoff and AI-state files |
| `handoffs/` | Датированные задания внешним помощникам / dated external-assistant handoffs |
| `scratch/` | Диагностические выгрузки вспомогательных скриптов / diagnostic exports from helper scripts |

Архивные документы могут намеренно содержать числа, формулировки и пути,
которые были верны на момент их создания и позднее изменились.

Archived documents may intentionally retain counts, wording, and paths that
were accurate when written and subsequently changed.
